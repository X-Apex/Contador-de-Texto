# Contador de Texto

Este pequeño ejercicio se encuentra programado en **C++**, **Java** y **Python** que permiten contabilizar:
- Número de **caracteres** (sin contar espacios en blanco, tabulaciones ni saltos de línea).
- Número de **palabras**.
- Número de **oraciones** (considerando los signos `.`, `?`, `!`).

---

## 📌 Objetivo
El objetivo es practicar el manejo de cadenas de texto con 'char' y comparar la sintaxis en distintos lenguajes de programación.
